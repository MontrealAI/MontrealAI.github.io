Welcome to the installation page for AGI Unicorn v0. Follow the instructions below to download and install the setup script on your machine. AGI Unicorn brings an unparalleled AI experience directly to your system, fully aligned with the powerful and grand vision of AGI.Eth.

Installation Instructions

Step 1: Ensure Docker is installed and running

If Docker is not already installed on your machine, follow the instructions for your operating system:

  • Mac OS X: Install Docker Desktop for Mac
  • Ubuntu: Install Docker Engine on Ubuntu
  • Windows: Install Docker Desktop for Windows

After installing Docker, make sure it's running:

docker --version

Step 2: Download and Run the Setup Script

Once you've downloaded setupagiunicornv0ii.sh, you can run it using the following command:

chmod +x setupagiunicornv0ii.sh && ./setupagiunicornv0ii.sh

This script will:

  • Stop and remove any existing containers named ollama_unicorn_v0.
  • Pull the necessary Docker image if not already available.
  • Run the Docker container on a specified port.
  • Create the required model and setup the Flask application.

Follow the on-screen prompts to complete the installation.

Step 3: Access the AGI Unicorn Application

Once the setup is complete, you can access the AGI Unicorn application by navigating to http://localhost:5000 in your web browser.

If you encounter any issues during installation, please ensure that Docker is properly configured and running.
