Welcome to the future of companionship. Follow the instructions below to download and install AGI Boyfriend on your machine.

Download setupagiboyfriendv0.sh

Installation Instructions:

For Mac OS X and Ubuntu:

1. Ensure Docker is installed and running:

brew install docker

or

sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io

2. Download the setup script:

curl -O https://montrealai.github.io/setupagiboyfriendv0.sh

3. Make the script executable:

chmod +x setupagiboyfriendv0.sh

4. Run the setup script:

./setupagiboyfriendv0.sh

For Windows:

1. Ensure Docker Desktop is installed and running. Download it from here.

2. Download the setup script:

Invoke-WebRequest -Uri "https://montrealai.github.io/setupagiboyfriendv0.sh" -OutFile "setupagiboyfriendv0.sh"

3. Open Git Bash or WSL and navigate to the directory where the script is downloaded.

4. Make the script executable:

chmod +x setupagiboyfriendv0.sh

5. Run the setup script:

./setupagiboyfriendv0.sh

Follow the on-screen instructions to complete the setup.
