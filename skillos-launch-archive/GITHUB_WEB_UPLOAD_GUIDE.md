# GitHub Upload Guide

Upload this folder to:

```text
https://github.com/MontrealAI/MontrealAI.github.io
```

Folder to upload:

```text
skillos-launch-archive
```

Target public URL:

```text
https://montrealai.github.io/skillos-launch-archive/
```

## Steps

1. Open the MontrealAI.github.io repository.
2. Click Add file -> Upload files.
3. Drag the folder `skillos-launch-archive` into GitHub.
4. Commit with message: `Add SkillOS public launch archive`.
5. Open https://montrealai.github.io/skillos-launch-archive/.
