# SkillOS Pilot 001 Public Plan

## Purpose

Pilot 001 asks whether SkillOS can make one practical workflow cheaper, faster, and better while keeping humans in control.

## Workflow

Sales follow-up emails from call notes.

## Rule

```text
Agent drafts. Human approves. Nothing is sent automatically.
```

## Size

50-100 real jobs.

## Metrics

- Human baseline time per job
- SkillOS-assisted time per job
- Human edit time
- Accepted draft rate
- Quality score before and after
- Missing next-step issues
- Unsupported commitment issues
- Unauthorized actions
- Skill version used
- Lessons captured

## Publishing posture

Only safe, aggregated results should be published. Do not publish private call notes, customer names, personal data, or confidential business context.
