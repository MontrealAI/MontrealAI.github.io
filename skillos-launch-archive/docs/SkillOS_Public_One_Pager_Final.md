# SkillOS Public One-Pager

## One line

**SkillOS is an open-source proof that AI-agent work can compound.**

## The loop

```text
Job -> Trace -> Lesson -> Tested Skill -> Better Future Work
```

## Why it matters

A normal agent completes work. SkillOS turns work into reusable capability.

## What to open

- Live proof: https://montrealai.github.io/skillos/
- GitHub: https://github.com/MontrealAI/skillos

## Pilot 001

50-100 real sales follow-up jobs. Agent drafts. Human approves. Nothing is sent automatically.

## Boundary

This public archive contains no private customer data, no private commercial terms, no financing materials, no confidential internal strategy, and no sensitive internal arrangements. Current metrics are reference-workflow results, not audited customer results or guarantees of future outcomes.
