Proof‑First Ascension

No value without evidence.
No autonomy without authority.
No settlement without validation.

Agents can act, but value moves only when evidence survives validation. The archive preserves what worked so future mandates start stronger. This is the difference between raw automation and governed compounding capability.

Job Spec→Artifact→Test→Hash→Attestation→Validation→Settlement→Archive

Job Spec

Goal, scope, inputs, acceptance criteria, authority.

Artifact

Code, report, data, workflow, model output, or operational result.

Test

Automated or human-reviewed criteria proving the artifact satisfies the mandate.

Hash

Tamper-evident record of the work product and proof bundle.

Attestation

Signed statement by agent, validator, node, or council.

Validation

Evidence review, replay, challenge, or approval path.

Settlement

Value movement only after proof passes.

Archive

Reusable recipe becomes institutional memory.

AGI.Eth Namespace Grammar

One rule. Scoped meaning. Machine-readable authority.

Main grammar

<entity>.( <env>. )<role>.agi.eth

role ∈ {club, agent, node}
env ∈ ENV_SET, optional; examples: alpha, x, …

Validator / club

alice.club.agi.eth

alice.alpha.club.agi.eth

Agent

helper.agent.agi.eth

helper.alpha.agent.agi.eth

Node

gpu01.node.agi.eth

gpu01.alpha.node.agi.eth

Sovereigns / AGI Businesses

Global sovereign: <sovereign>.agi.eth

Ecosystem root: env.agi.eth (examples: alpha.agi.eth, x.agi.eth)

AGI Business: <business>.env.agi.eth (example: ops.alpha.agi.eth)

Scope rule and package

entity.env.role.agi.eth is recognized and developed only within env.agi.eth.

Official package:

env.agi.eth · env.agent.agi.eth · env.node.agi.eth · env.club.agi.eth

Optional aliases: agent.env.agi.eth · node.env.agi.eth · club.env.agi.eth. Optional aliases are env-local and not official by default unless explicitly whitelisted.
