AGI.Eth Namespace Grammar

Main grammar

<entity>.( <env>. )<role>.agi.eth
role ∈ {club, agent, node}
env optional: alpha, x, …

Validator / club

alice.club.agi.eth
alice.alpha.club.agi.eth

Agent

helper.agent.agi.eth
helper.alpha.agent.agi.eth

Node

gpu01.node.agi.eth
gpu01.alpha.node.agi.eth

Sovereigns

<sovereign>.agi.eth
env.agi.eth → alpha.agi.eth, x.agi.eth
<business>.env.agi.eth → ops.alpha.agi.eth

Scope rule

entity.env.role.agi.eth is recognized and developed only within env.agi.eth.

Official package

env.agi.eth · env.agent.agi.eth · env.node.agi.eth · env.club.agi.eth

Aliases

agent.env.agi.eth · node.env.agi.eth · club.env.agi.eth
Aliases are env-local and not official by default unless whitelisted.
