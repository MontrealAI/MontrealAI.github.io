AGI.Eth Namespace Grammar

One rule. Scoped meaning. Machine-readable authority.

<entity>.( <env>. )<role>.agi.eth

role ∈ {club, agent, node} · env ∈ ENV_SET, optional; examples: alpha, x, …

Examples

Validator / club:

alice.club.agi.ethalice.alpha.club.agi.eth

Agent:

helper.agent.agi.ethhelper.alpha.agent.agi.eth

Node:

gpu01.node.agi.ethgpu01.alpha.node.agi.eth

Sovereigns / AGI Businesses

Global sovereign: <sovereign>.agi.eth

Ecosystem root: env.agi.eth examples: alpha.agi.eth x.agi.eth

AGI Business: <business>.env.agi.eth example: ops.alpha.agi.eth

Scope rule: entity.env.role.agi.eth is recognized and developed only within env.agi.eth.

Official environment package

env.agi.ethenv.agent.agi.ethenv.node.agi.ethenv.club.agi.eth

Optional aliases

agent.env.agi.ethnode.env.agi.ethclub.env.agi.eth

Optional aliases are env-local and not official by default unless explicitly whitelisted.

The Three Clocks of Machine-Speed Ascension

Autonomy without governance collapse

Fast clock — execution

Agents decompose, call tools, run jobs, test outputs, retry failures, and package artifacts.

Medium clock — proof / settlement

Proof bundles, validation, receipts, challenges, and settlement.

Slow clock — governance / promotion

Policy, disputes, upgrades, sovereign promotion, and archive law.

Machine speed belongs in execution. Trust belongs in proof. Permanence belongs in governance.

Proof-First Doctrine

  • No value without evidence.
  • No autonomy without authority.
  • No settlement without validation.

The system is designed so that agents can act, but value only moves when evidence survives validation. The archive preserves what worked so future mandates start stronger.

Exploration proposes. Proof verifies. Settlement moves value. Governance decides what becomes law.
