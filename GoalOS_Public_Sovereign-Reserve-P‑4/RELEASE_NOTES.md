# GoalOS Public Institutional Edition P-4.0

This release supersedes P-3.0 and P-3.1. The earlier deployment omitted required website assets, causing unstyled local rendering. P-4.0 removes that failure mode by embedding critical CSS and JavaScript directly into the primary pages while retaining developer-readable mirrors.

The release is public-safe and claim-bounded. It is not an investment offer, token sale, production certification, or transfer of ownership.
