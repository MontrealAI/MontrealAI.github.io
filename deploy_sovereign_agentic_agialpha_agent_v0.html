Download & Deploy the SOVEREIGN AGENTIC AGI ALPHA AGENT (deploy_sovereign_agentic_agialpha_agent_v0.sh) Script

To set up and deploy the Sovereign Agentic AGI ALPHA environment on your own device or server, simply download the script below. No wallet connection or $AGIALPHA verification is required just to download.

Step 1: Click the button below to download deploy_sovereign_agentic_agialpha_agent_v0.sh.

Download Deploy Script

Step 2: Open a terminal where you saved deploy_sovereign_agentic_agialpha_agent_v0.sh and make it executable (on Linux/macOS):

chmod +x deploy_sovereign_agentic_agialpha_agent_v0.sh

Step 3: Run the script:

./deploy_sovereign_agentic_agialpha_agent_v0.sh

This script will:

  • Create a sovereign_agent_deploy directory with all necessary files.
  • Build Docker containers (ensuring cryptography is installed and pinned versions of libraries).
  • Prompt you to select a local LLM or use your cloud keys if you have them.
  • Launch a minimal Flask-based web UI at http://localhost:5000 for browsing & chat.
  • Attach to the Docker container for a Terminal-based chat (type exit or quit to end).

Note: The script includes logic to detect your $AGIALPHA token holdings if you want advanced usage. That token verification is part of the deployment process, not this download.

On Windows, use WSL or Git Bash to run the script. Ensure Docker Desktop is installed.
