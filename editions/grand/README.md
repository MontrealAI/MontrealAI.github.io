# GoalOS Partner MASTERCLASS - Grand Institutional Edition

## Start

Open `START_HERE.html` after extracting the package.

Everything is designed to run locally. The interactive experiences do not require an account, backend, API key, remote font, script, image, or network connection. An optional user-configured AI endpoint remains disabled unless the user explicitly configures it.

## What this edition is

This is the complete partner-facing institution for presenting, teaching, operating, challenging, diligencing, and designing GoalOS.

It combines two complementary front doors:

1. **Grand Institutional Edition** - the complete operating masterclass with Proof Theatre, curriculum, Proof Mission, AI Council, stress theatre, governed RSI frontier, Merkle lab, diligence vault, Deal Architect, value lab, coach, assessment, and exports.
2. **APEX Boardroom Edition** - a premium visual operating theatre for high-caliber executive and strategic partner encounters.

## Recommended use

### 90 seconds

Open the Grand Institution and select `90-second proof theatre`.

### 7 minutes

Use the Grand Institution's boardroom presentation mode, then show one mission gate and the tamper test.

### 45 minutes

Use the editable deck through the authority, proof, RSI, and partner sections. Open the live institution for one mission and one stress incident.

### 90 minutes

Run the full interactive masterclass: curriculum, Proof Mission, Chronicle, RSI Frontier, Merkle proof, diligence, and Partner Charter.

### Three hours

Use the Fieldbook to freeze a real Mission Contract, Proof Debt register, validator route, rollback policy, Mission 2 design, and 30/60/90 Partner Charter.
