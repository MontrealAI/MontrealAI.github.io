Welcome to the AGI Cat Setup

To get started with your AGI Cat setup, follow the instructions below. This guide will walk you through the installation process on Mac OSX, Ubuntu, and Windows. Start by downloading the setup script:

Download setupagicatv1.sh

Installation Instructions

Mac OSX and Ubuntu

1. Ensure you have Docker installed. If not, install it by following the instructions here.

2. Open your terminal.

3. Navigate to the directory where the script is downloaded:

cd /path/to/downloaded/file

4. Make the script executable:

chmod +x setupagicatv1.sh

5. Run the script:

./setupagicatv1.sh

Windows

1. Ensure you have Docker installed. If not, install it by following the instructions here.

2. Open PowerShell as an administrator.

3. Navigate to the directory where the script is downloaded:

cd C:\path\to\downloaded\file

4. Run the script:

sh setupagicatv1.sh

Troubleshooting

If you encounter any issues during the installation, ensure that Docker is correctly installed and running. For further assistance, refer to the Docker documentation or contact our support team.
