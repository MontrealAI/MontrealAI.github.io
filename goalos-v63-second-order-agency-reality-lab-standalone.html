Public‑alpha boundary

This interface is a local demonstration and learning lab. It prepares proof objects. It does not execute live external work or move value.

  • No backend.
  • No project data sent to GoalOS.
  • No wallet in local mode.
  • No transaction in local mode.
  • No user funds.
  • No live settlement.
  • No investment advice.
  • No production authority.
  • Human review required for high-impact outcomes.
  • Claim boundaries remain explicit.
