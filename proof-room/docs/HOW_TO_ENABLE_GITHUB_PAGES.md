# How to enable GitHub Pages

1. Open the `proof-room` repository.
2. Click **Settings**.
3. In the left menu, click **Pages**.
4. Under **Build and deployment**, choose:

```text
Deploy from a branch
```

5. Select:

```text
Branch: main
Folder: /root
```

6. Click **Save**.

The site will be published after GitHub finishes deploying it.
