# What makes the QUEBEC.AI Proof Room convincing

The strongest public proof is simple enough for non-technical readers and structured enough for serious reviewers.

## The convincing pattern

```text
Here is the mission.
Here are the AI agent roles.
Here is how the work was bounded.
Here is how the output was checked.
Here is the evidence.
Here is what became reusable.
Here is how the next cycle starts stronger.
```

## Why the Proof Room matters

The Manifesto says what QUEBEC.AI believes.

The Proof Room shows how QUEBEC.AI works.

## What a reviewer can inspect

```text
Claims matrix
Mission definition
Agent constellation
Validation checklist
ProofBundle
Capability Archive
Replay and review notes
Reviewer honesty box
```

## What should be added over time

The first public repository is the proof layer.

The next step is to add more real dockets:

```text
ED‑002 — AI‑First Website Intelligence Workflow
ED‑003 — AI‑First Media Command Center
ED‑004 — AI‑First Security and Governance Review
ED‑005 — AI‑First Enterprise Transformation Case
```

Each docket should be clear, bounded, public-safe, and reviewable.
