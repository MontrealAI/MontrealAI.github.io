# Data Processing Addendum — Counsel-Review Template

**Status:** Requires privacy counsel and completion for the actual roles and jurisdictions.

Identify controller/business and processor/service-provider roles; subject matter; duration; data subjects; categories; purposes; documented instructions; confidentiality; security measures; subprocessors; cross-border transfer mechanism; assistance; breach notice; audits; deletion/return; government requests; conflict and signatures.

Public GitHub Pages, public IPFS gateways, public blockchain records, and client-side wallet signatures are not appropriate storage for confidential personal data.
