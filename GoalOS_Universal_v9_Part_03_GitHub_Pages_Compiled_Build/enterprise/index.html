Deployment choices

Public, private, or local

  • Public/static: public preview, public documentation, and AGI Club member verification.
  • Private enterprise: host the static build behind the organization’s IdP, VPN, authenticated reverse proxy, or controlled device management.
  • Local/offline: download the single HTML file and open it locally; the Money Machine itself makes no network requests.
Browser-local data posture. The standalone Money Machine stores working state in the browser and contains no embedded analytics, wallet integration, or server API calls.
