# Security

Report security concerns to `secretariat@montreal.ai` with release, URL, time, browser, and reproducible details. Do not send private keys, seed phrases, passwords, or unnecessary confidential data.

The member signature must never request a transaction, token approval, seed phrase, private key, or asset permission. Verify the page URL and message before signing.

See `docs/SECURITY_HEADERS.md`, `public/legal/SECURITY.html`, and `docs/PRODUCTION_ACTIVATION_CHECKLIST.md`.
