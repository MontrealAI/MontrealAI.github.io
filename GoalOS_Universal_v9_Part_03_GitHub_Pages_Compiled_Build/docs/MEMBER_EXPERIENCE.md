# Member Experience Standard

The member needs only:

- MetaMask; and
- the label of a direct `*.club.agi.eth` name owned by the selected MetaMask account.

The member does not need ETH, gas, a token approval, an API key, a GoalOS password, a custom RPC, a contract address, or technical ENS knowledge.

## Friendly errors

- Wrong label: return to the label field with an example.
- Wrong account: offer **Try another MetaMask account**.
- Cancelled signature: state that nothing changed and no asset action occurred.
- Pending MetaMask request: ask the member to finish or close the existing request.
- Expired wrapped name: display the wrapper expiry and route to support.
- RPC outage: explain that Ethereum is temporarily unavailable and no wallet action occurred.
