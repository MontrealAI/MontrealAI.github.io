# GoalOS Universal Production System v9 — Five-Minute Quick Start

## Choose the operating mode

1. **AGI Club member** — deploy the static build, open `agi-club.html`, enter the short label such as `elite`, connect MetaMask, and approve one free signature. Current direct `elite.club.agi.eth` ownership is checked on Ethereum Mainnet.
2. **Enterprise team** — open `enterprise/index.html`; select the data classification; accept the enterprise boundaries; launch the browser-local workspace or download the one-file application.
3. **Local/offline user** — open `local/index.html`, download `GoalOS_Autonomous_Money_Machine_OMEGA_Offline.html`, and open it in a modern browser.
4. **Public visitor** — use `buyer-funnel.html` without a wallet or account.
5. **Operator** — use `operator/index.html`, the live wallet acceptance console, QA report, and activation checklist.

## Deploy

- GitHub Pages: push the source package to a repository, enable Pages through GitHub Actions, and run the included workflow.
- IPFS: build, export `ipfs-export/`, and pin the directory/CAR with at least two independent providers when practical.
- Private enterprise: host the static build behind the organization’s IdP, VPN, authenticated reverse proxy, or managed-device controls.
- Local: no hosting is required.

## Production activation

Do not label the deployment production-active until the live direct-ENS owner cases, legal approval, security approval, support contacts, hashes, and recovery plan are recorded.
