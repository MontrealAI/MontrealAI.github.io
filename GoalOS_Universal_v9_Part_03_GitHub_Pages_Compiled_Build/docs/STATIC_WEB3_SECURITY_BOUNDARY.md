# Static Web3 Security Boundary

## What is enforced

- MetaMask account control through a locally verified signature.
- Exact direct-name syntax.
- Current ENS Registry or Name Wrapper ownership.
- Wrapped-name expiry.
- Periodic ownership revalidation.
- Versioned legal-bundle acknowledgement.

## What is not secret

A static application hosted on GitHub Pages or a public IPFS gateway delivers code and files to the browser. A determined party may retrieve those bytes even when the intended interface is gated. This release provides Web3 entitlement and member UX—not DRM.

Confidential member assets require a separately reviewed encrypted distribution system with decentralized or operator-controlled key release.

See `ENCRYPTED_MEMBER_DISTRIBUTION_BLUEPRINT.md` for the next confidentiality layer. The blueprint is deliberately not represented as activated.
