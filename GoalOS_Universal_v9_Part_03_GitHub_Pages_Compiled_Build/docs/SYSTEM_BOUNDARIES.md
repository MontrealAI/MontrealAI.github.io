# System Boundaries

- **Proof is not truth by itself.** Evidence quality, reviewer competence, scope, specifications, incentives, and delayed outcomes remain material.
- **A wallet signature is not KYC or legal identity.** It proves control of an account for the signed message.
- **ENS ownership is not a security, equity, beneficiary, profit, governance, or income right.** It activates only the permissions expressly stated by the applicable system and agreement.
- **Public static gating is not DRM or confidentiality.** Delivered files can be copied.
- **Financial models are scenarios, not guarantees.** Replace assumptions with observed evidence.
- **Local storage is not enterprise records management.** Export, back up, retain, and delete according to policy.
- **Professional and consequential decisions require qualified review.** GoalOS does not silently authorize deployment, payment, contracting, settlement, or publication.
