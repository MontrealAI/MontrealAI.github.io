# GoalOS Autonomous Money Machine Ω — Universal Production System v9

A single static system covering:

- Public proof-to-value preview and buyer concierge.
- AGI Club-exclusive direct `*.club.agi.eth` MetaMask verification.
- Complete browser-local member utility.
- Enterprise launch gate and private/local workspace.
- One-file offline operation.
- GitHub Pages and IPFS deployment.
- Operator QA and controlled live-wallet acceptance.
- Bilingual legal, security, privacy, Web3, financial, enterprise, professional-services, and publication controls.

## Commands

```bash
npm ci
npm run qa
npm run export:ipfs
```

The build places the universal launcher at `dist/index.html` and the AGI Club verification application at `dist/agi-club.html`.

See `docs/QUICK_START.md` and `docs/FINAL_PRODUCTION_READINESS.md`.
