Welcome to the installation page for AGI Boyfriend Node v0. Follow the instructions below to download and install the setup script on your machine. AGI Boyfriend Node v0 is designed to be fully aligned with the powerful and grand vision of AGI.Eth.

Installation Instructions

Step 1: Ensure Docker is installed and running

If Docker is not already installed on your machine, follow the instructions for your operating system:

  • Mac OS X: Install Docker Desktop for Mac
  • Ubuntu: Install Docker Engine on Ubuntu
  • Windows: Install Docker Desktop for Windows

After installing Docker, ensure it is running by checking the version:

docker --version

Step 2: Download and Run the Setup Script

Once you've downloaded setupagiboyfriendnodev0.sh, run the following commands to set up the node:

  • Mac OS X & Ubuntu:
chmod +x setupagiboyfriendnodev0.sh && ./setupagiboyfriendnodev0.sh
  • Windows (using Git Bash or WSL):
bash setupagiboyfriendnodev0.sh

This script will:

  • Stop and remove any existing containers named ollama_boyfriend_v0.
  • Pull the necessary Docker image if not already available.
  • Run the Docker container on port 11440.
  • Create the required model and set up the Flask application.

Step 3: Access the AGI Boyfriend Node Application

Once the setup is complete, access the AGI Boyfriend Node application by navigating to http://localhost:5000 in your web browser.

If you encounter any issues during installation, please ensure that Docker is properly configured and running. You can find additional help in the official Docker documentation.
