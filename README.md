## Governed recursive self-improvement

The RSI laboratory implements the operating sequence:

`TARGET → EMIT → FILTER → ATLAS → TEST-PLAN → EVAL → INSERT → PROMOTE`

Exploration may guide allocation. Outcome authority remains mechanical. Promotion requires executed evidence, baseline advantage, replay, stress, persistence, validator independence, scoped authority and rollback.

The decisive compounding test is Mission 2: does an admitted, scoped capability improve fresh work under equal constraints? No Mission 2, no compounding claim.
