# LaTeX edition

Compile with:

```bash
./build.sh
```

Requirements: XeLaTeX, latexmk, Noto Serif, Inter, and Noto Sans Mono. The manuscript body is in `body.tex`; `paper.tex` provides the institutional typesetting system and full-page cover. `references.bib` is supplied as a machine-readable bibliography companion; the paper's visible reference list remains embedded in the manuscript for parity with the DOCX edition.
