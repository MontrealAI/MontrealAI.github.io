# Operator Acceptance Kit

- `LIVE_WALLET_ACCEPTANCE_CONSOLE.html` — plain-English test recorder and JSON report exporter.
- `LIVE_WALLET_ACCEPTANCE_RUNBOOK.md` — step-by-step live MetaMask and ENS acceptance procedure.
- `LIVE_ACCEPTANCE_CASES.example.json` — safe placeholder matrix for the seven required ownership cases.
- `../scripts/verify-live-membership.ts` — optional command-line Mainnet ownership check using the same Registry/Wrapper rules as the application.

The operator kit contains no wallet keys and must never be populated with seed phrases or private keys.
