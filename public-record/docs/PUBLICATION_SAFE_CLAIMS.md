# Publication-safe claim guidance

Use:
- Public record
- Evidence layer
- Proof architecture
- Strategic reference library
- Public build surface
- Prior-art record
- Founder-led articles
- Proof Room
- Evidence Dockets
- ProofBundles
- Claim boundaries

Avoid:
- achieved AGI
- achieved ASI
- guaranteed returns
- empirical SOTA without benchmarked evidence
- autonomous legal or governmental authority
- unrestricted autonomy claims
