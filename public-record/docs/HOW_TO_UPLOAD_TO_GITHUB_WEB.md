# How to upload this package with the GitHub web interface

1. Download the ROOT FOLDER zip.
2. Unzip it on your computer.
3. You will see a folder named `public-record`.
4. Go to `https://github.com/QuebecAI/QuebecAI.github.io`.
5. Click **Add file → Upload files**.
6. Drag the entire `public-record` folder into GitHub.
7. Commit message: `Publish QUEBEC.AI Public Record`.
8. Click **Commit changes**.
9. Wait a few minutes.
10. Open `https://QuebecAI.github.io/public-record/`.

Important: do not upload the zip itself. Unzip first.

Correct path: `public-record/index.html`.

Wrong path: `public-record/public-record/index.html`.
