# How to update the Public Record

The main page is `index.html`.
The searchable data file is `data/public-record-index.json`.

To add a new LinkedIn article:

1. Open `index.html`.
2. Find the section with `article-card` entries.
3. Copy one card.
4. Change the number, title, category, description, and link.
5. Also add the item to `data/public-record-index.json`.

Recommended update cadence: after every major LinkedIn Featured update or after every major QUEBEC.AI publication.
