# Recette de déploiement / Live deployment acceptance

## Portes obligatoires / Required gates

- [ ] Le commit de lancement a été revu et signé par une personne autorisée. / Launch commit reviewed and approved by an authorized person.
- [ ] L’ancien `index.html` et le commit précédent sont identifiables pour retour arrière. / Previous root page and commit are identifiable for rollback.
- [ ] Les 28 pages françaises et les 28 pages anglaises répondent en HTTPS. / All 28 French and 28 English pages respond over HTTPS.
- [ ] Les alternates `hreflang`, canoniques, sitemap, robots et carte sociale sont corrects. / Hreflang, canonicals, sitemap, robots and social card are correct.
- [ ] Le menu mobile, les images, les téléchargements et les anciennes URL ont été testés. / Mobile menu, images, downloads and legacy URLs were tested.
- [ ] Aucun formulaire serveur, compte, analyse comportementale, portefeuille ou paiement n’a été ajouté au site public. / No server form, account, behavioural analytics, wallet or payment was added to the public site.
- [ ] L’entité, le courriel, les mentions de marque et les textes juridiques correspondent à l’opérateur réel. / Entity, email, branding and legal text match the actual operator.
- [ ] Les conseillers ont examiné les marchés, services, données et juridictions réellement activés. / Counsel reviewed the markets, services, data and jurisdictions actually activated.
- [ ] Les affirmations publiques restent dans la Frontière des affirmations. / Public claims remain within the Claim Boundary.
- [ ] Les téléchargements ont été vérifiés par SHA-256 et analysés avant publication. / Downloads were verified by SHA-256 and reviewed before publication.
- [ ] Le plan d’incident, le responsable et le canal de retrait d’urgence sont connus. / Incident plan, owner and emergency takedown channel are known.

## Décision / Decision

- [ ] **ACTIVER / ACTIVATE**
- [ ] **CONDITIONNER / CONDITION**
- [ ] **RÉPARER / REPAIR**
- [ ] **ARRÊTER / STOP**
