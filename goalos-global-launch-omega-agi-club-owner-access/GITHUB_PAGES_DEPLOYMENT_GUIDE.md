# GitHub Pages Deployment Guide — GoalOS Global Launch Ω v11.0.0-GL1

## Publish

1. Unzip the deployment ZIP locally.
2. Copy the root `.nojekyll` file and the `goalos-global-launch-omega-agi-club-owner-access/` directory into the root of `MontrealAI/MontrealAI.github.io` on the Pages publishing branch.
3. Commit and push.
4. Open:

`https://montrealai.github.io/goalos-global-launch-omega-agi-club-owner-access/`

Do not upload the ZIP itself as the website. Preserve the directory structure.

## Access model

The application is a static, public-source GitHub Pages institution. Ordinary use is gated by current direct ownership of an exact one-label `club.agi.eth` name. The gate is a conditional-use, evidence and community-access mechanism—not confidential DRM or server-side authorization.
