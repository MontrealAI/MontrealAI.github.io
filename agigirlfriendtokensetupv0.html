Installation Instructions

Step 1: Ensure Docker is installed and running

If Docker is not installed, follow the instructions for your system:

  • Mac OS X: Install Docker Desktop
  • Ubuntu: Install Docker Engine
  • Windows: Install Docker Desktop

Verify Docker installation:

docker --version

Step 2: Download and Run the Setup Script

After downloading setuptokenagigirlfriendv0.sh, run:

chmod +x setuptokenagigirlfriendv0.sh && ./setuptokenagigirlfriendv0.sh
  • Stops any running containers named ollama_girlfriend_v0.
  • Pulls the required Docker image.
  • Runs the application on your local machine.

Step 3: Access the Application

Once installed, open http://localhost:5000 in your browser.
